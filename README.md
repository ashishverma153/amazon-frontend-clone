# Amazon Frontend Clone

A responsive, high-fidelity replica of the Amazon e-commerce homepage layout built using semantic frontend web standards.

## 🛠️ Tech Stack & Architecture
- **HTML5:** Structured with semantic elements for better layout hierarchy and accessibility.
- **CSS3 Layouts:** Built using modern Flexbox configurations to mimic Amazon's multi-tier navigation headers, dynamic product grids, and multi-column footers.
- **Responsive Styling:** Hand-crafted transitions, dropdown hover states, and smooth UI scaling.
